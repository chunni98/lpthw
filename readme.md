> 学习本书过程中不要使用集成开发环境（IDE）。依赖 IDE 的结果就是没办法使用新的编程语言，因为你要等着企业卖给你一个支持这门语言的 IDE，但除非已经有了很多人在使用这门语言，否则企业是不会为它开发 IDE 的。如果你有信心使用 Vim、Emacs、Atom 之类的程序员的文本编辑器写代码，那你就不必等待第三方推出 IDE 了，尽管有些场合下 IDE 也不错，比如针对已有的庞大代码库，但如果使用 IDE 上了瘾，你的个人前途就会受限。
> 
> —— Zed A. Shaw
